# Rune OS ALPHA

## Introduction

Rune OS 3 is an official continuation of the Rune OS Python-based operating system.

## How to use

1. Download Rune OS 3
2. Run `main.py`

Bug reporting will be much appreciated.

This OS can:

* manage accounts with passwords
* support apps*
* and much more!
